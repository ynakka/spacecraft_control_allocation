# Spacecraft Control Allocation
A python code for testing control allocation for spacecraft simulator dynamics.

### Dependencies: 
1) Numpy 
2) Scipy
3) Math
4) Matplotlib
5) Cvxpy  

### Control Allocation Methods Included
1) Psuedo-inverse 
2) QP formulation 
3) CLF QP formulation
